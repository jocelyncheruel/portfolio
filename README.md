Portfolio personnel présentant mon parcours, mes projets personnels et mes projets IUT.

## Aperçu

Ce portfolio met en avant :

- mes projets personnels
- mes projets académiques liés au BUT Science des Données
- mes compétences techniques
- mon parcours

L'interface inclut notamment :

- un site bilingue `FR / EN`
- un changement de thème `clair / sombre`
- une galerie d'images pour les projets
- un zoom sur les visuels
- un affichage PDF intégré pour le CV et les lettres de recommandation
